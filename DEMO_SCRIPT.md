# FoxBot AI Chatbot Demo Script

## 60?90 Second Demo Script

Hey everyone, this is FoxBot AI Chatbot, my Blaze Builder Challenge project.

FoxBot is a Blaze-connected creator assistant for livestreamers. It connects to Blaze, listens to live chat, and responds to viewer commands in real time.

The goal is to help creators keep chat active with giveaways, FoxCoins, mini games, rewards, boss battles, custom commands, and OBS overlays.

First, I?ll open the Judge Demo page and run !help so you can see the command system.

Next, I?ll claim daily FoxCoins and run !foxhunt, which lets viewers earn points during the stream.

Then I?ll open the reward shop with !shop and redeem a reward with !redeem hug.

That redemption also appears on the OBS Redemptions Overlay, so stream viewers can see the action live.

FoxBot also has a boss battle system. I can start a boss with !startboss Cyber Fox Dragon, then viewers can attack with !attack or spend FoxCoins on !powerattack.

The Boss Battle Overlay shows live HP, top damage, and commands for chat.

There is also a Giveaway Overlay for stream giveaways, an Economy Dashboard for managing points and rewards, and a Smoke Test page to make sure major commands are working before going live.

FoxBot is more than a chatbot. It is a full Blaze creator engagement platform with live chat tools, overlays, games, and viewer rewards.
